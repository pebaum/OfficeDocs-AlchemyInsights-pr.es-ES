# <a name="public-folder-access-from-mobile-devices"></a>Acceso a carpetas públicas desde dispositivos móviles

Si desea tener una funcionalidad similar a la de las carpetas públicas y que el contenido sea accesible en dispositivos móviles, consulte [información sobre los grupos de Office 365](https://support.office.com/article/learn-about-office-365-groups-b565caa1-5c40-40ef-9915-60fdb2d97fa2) para obtener una mejor alternativa.
