# <a name="activate-a-new-subscription-with-a-product-key"></a>Activar una suscripción nueva con una clave de producto

1. Para activar una nueva suscripción con la clave de producto, vaya [https://office.com/setupkey](https://office.com/setupkey)a.

2. Escriba la clave de producto y siga las instrucciones para activar la suscripción. Si tiene problemas con la activación, consulte esta [lista de los errores comunes de la clave de producto y sus soluciones](https://docs.microsoft.com/office365/admin/misc/product-key-errors-and-solutions).
