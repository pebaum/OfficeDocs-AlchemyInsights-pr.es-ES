# <a name="bloquear-o-desbloquear-remitentes-en-outlookcom"></a>Bloquear o desbloquear remitentes en Outlook.com

Aceptar bloquear los mensajes de las direcciones de correo electrónico específica o dominio llegue a la bandeja de entrada. Cuando agrega una dirección de correo electrónico o un dominio a la lista de remitentes bloqueados, Outlook.com mueve automáticamente los mensajes de ese, remitente a la moqueta correo electrónico no deseado.

Las instrucciones hijo ligeramente de dependiendo de si está utilizando la versión beta de Outlook.com. Elegir qué versión de Outlook.com usa para ver los [pasos que le conciernen](https://support.office.com/es-es/article/remitentes-de-bloquear-o-desbloquear-remitentes-en-outlook-com-afba1c94-77bb-4f50-8b85-057cf52f4d5e?ui=es-ES&rs=es-ES&ad=ES).
