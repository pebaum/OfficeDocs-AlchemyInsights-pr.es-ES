# <a name="delete-a-folder"></a>Eliminar una carpeta

1. En el panel izquierdo, haga clic en la carpeta o subcarpeta que desee eliminar y seleccione **Eliminar**. 
    
2. Seleccione **Aceptar** para confirmar. 
    
Para obtener más información, vea [Eliminar una carpeta en Outlook.com](https://go.microsoft.com/fwlink/p/?linkid=873134).
