# <a name="creating-inbox-rules-for-shared-mailboxes"></a>Creación de reglas de bandeja de entrada para buzones compartidos

Puede agregar reglas a un buzón compartido al que tenga acceso de forma similar a como se agregan las reglas a su propia cuenta.
  
Asegúrese de que es miembro del buzón compartido en cuestión y, a continuación, siga los pasos de este artículo: [agregar reglas a un buzón compartido](https://support.office.com/article/b0963400-2a51-4c64-afc7-b816d737d164)
