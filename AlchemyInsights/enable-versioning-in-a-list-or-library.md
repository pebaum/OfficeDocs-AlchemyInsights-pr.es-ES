# <a name="enable-versioning-for-a-list-or-library"></a>Habilitar el control de versiones para una lista o biblioteca

1. Vaya a la lista o biblioteca.
    
2. Haga clic en **configuración**y, a continuación, en configuración de **biblioteca** o de **lista**.
    
3. Haga clic en **configuración de versiones**.
    
4. Seleccione la configuración que desee y, a continuación, haga clic en **Aceptar**.
