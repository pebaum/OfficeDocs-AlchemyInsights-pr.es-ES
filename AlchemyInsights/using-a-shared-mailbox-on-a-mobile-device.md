# <a name="using-a-shared-mailbox-on-a-mobile-device"></a>Uso de un buzón compartido en un dispositivo móvil

Para poder usar un buzón compartido en el dispositivo móvil, debe [descargar Outlook Mobile](https://products.office.com/outlook-mobile-for-android-and-ios) y, a continuación, [Agregar un buzón compartido a Outlook Mobile](https://support.office.com/article/Add-a-shared-mailbox-to-Outlook-mobile-f866242c-81b2-472e-8776-6c49c5473c9f).
  
Para obtener ayuda: [obtener ayuda en la aplicación para Outlook para iOS y Android](https://support.office.com/article/Get-in-app-help-for-Outlook-for-iOS-and-Android-218a22d1-9fa5-4889-b689-de1c63493243)
