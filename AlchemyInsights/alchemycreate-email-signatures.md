# <a name="create-email-signatures"></a>Crear firmas de correo electrónico

Estos temas le ayudarán a crear las firmas de correo electrónico:
  
- Crear una firma para sus propios mensajes de correo electrónico: [crear y agregar una firma a los mensajes](https://support.office.com/article/8ee5d4f4-68fd-464a-a1c1-0e1c80bb27f2.aspx) o [crear y agregar una firma de correo electrónico en Outlook Web App](https://support.office.com/article/0f230564-11b9-4239-83de-f10cbe4dfdfc.aspx).
    
- Crear una firma de correo electrónico global para todos los mensajes enviados desde la empresa (también conocido como una renuncia de responsabilidad): [declinación de responsabilidades de mensaje de toda la organización, las firmas, pies de página o encabezados en Office 365](https://go.microsoft.com/fwlink/p/?linkid=391096).
