# <a name="manage-synchronized-user-accounts"></a>Administrar cuentas de usuario sincronizada

Normalmente, se utiliza el centro de administración de Office 365 para agregar y cambiar las cuentas de usuario. Sin embargo, no puede utilizarlo si está habilitada la sincronización de directorios o SSO. En su lugar, debe administrar los usuarios con Windows PowerShell tal como se describe en [Configurar propiedades de cuenta de usuario con Office 365 PowerShell](https://docs.microsoft.com/en-us/office365/enterprise/powershell/configure-user-account-properties-with-office-365-powershell ).
