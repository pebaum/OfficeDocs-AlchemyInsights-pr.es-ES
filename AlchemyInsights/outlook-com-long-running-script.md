# <a name="long-running-script"></a>Script de larga ejecución

Somos conscientes de los problemas con el "script de larga duración" en Outlook.com. Como solución alternativa, use la versión beta de Outlook.com.
  
[Obtenga más información sobre la versión beta de Outlook.com.](https://support.office.com/article/40676ad0-c831-45ac-a023-5be633be798d?wt.mc_id=Office_Outlook_com_Alchemy)
