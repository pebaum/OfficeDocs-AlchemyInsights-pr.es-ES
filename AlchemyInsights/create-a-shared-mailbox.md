# <a name="create-a-shared-mailbox"></a>Creación de un buzón compartido

Un buzón compartido permite que más de una persona envíe y reciba correo electrónico para un alias, como **sales@contoso.com**, sin asignar otra licencia.
  
Consulte [Create a Shared Mailbox](https://support.office.com/client/871a246d-3acd-4bba-948e-5de8be0544c9) para obtener más información.
