# <a name="learn-about-using-public-folders-in-outlook"></a>Obtener información sobre cómo usar carpetas públicas en Outlook

Las carpetas públicas de Outlook le proporcionan un gran espacio para recopilar, organizar y compartir información sobre temas o proyectos concretos dentro de la organización. Para obtener más información, vea [crear y compartir una carpeta pública en Outlook](https://support.office.com/article/a2835011-d524-4a5c-a207-05c159bb2a97).
