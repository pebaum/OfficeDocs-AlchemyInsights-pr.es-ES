# <a name="trying-to-remove-your-domain"></a>¿Está intentando quitar el dominio?

Siga los pasos en: [Quitar un dominio de Office 365](https://support.office.com/article/Remove-a-domain-from-Office-365-f09696b2-8c29-4588-a08b-b333da19810c.aspx).
  
Deberá actualizar los usuarios o grupos que estén asignados a dicho dominio para poder quitarlo. Consulte las [Preguntas frecuentes sobre los dominios](https://support.office.com/article/Domains-FAQ-1272bad0-4bd4-4796-8005-67d6fb3afc5a.aspx) para conocer las soluciones más habituales.
