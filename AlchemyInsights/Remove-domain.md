# <a name="trying-to-remove-your-domain"></a>¿Intenta quitar su dominio?

Para quitar el dominio de Office 365, siga los pasos de [quitar un dominio de Office 365](https://support.office.com/article/https://support.office.com/en-us/article/Remove-a-domain-from-Office-365-f09696b2-8c29-4588-a08b-b333da19810c.aspx).
  
Debe actualizar los usuarios o grupos que se asignan a ese nombre de dominio antes de quitar el dominio. Compruebe las [Preguntas más frecuentes de dominios](https://support.office.com/article/https://support.office.com/en-us/article/Domains-FAQ-1272bad0-4bd4-4796-8005-67d6fb3afc5a.aspx) para otras soluciones comunes.
