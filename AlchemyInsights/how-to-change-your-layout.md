# <a name="how-to-change-your-layout"></a>Cómo cambiar el diseño

Es fácil cambiar la configuración de Outlook.com para que tenga el aspecto y el comportamiento deseado. Seleccione **configuración** en la barra de herramientas para ver las opciones.
