# <a name="site-is-not-available"></a>Sitio no está disponible

Si su sitio de SharePoint o OneDrive no está disponible para varios usuarios, puede haber un problema de servicio temporal. Compruebe el[Panel de estado de servicio](https://admin.microsoft.com/AdminPortal/Home#/servicehealth) para ver si se ve afectado el inquilino.
