# <a name="setting-up-a-hybrid-environment"></a>Configurar un entorno híbrido

Para permitir que los usuarios colaboran sin ningún problema sin importar si sus datos se almacenan en servidores locales o en Office 365, es posible que necesite configurar un entorno híbrido. Para obtener una visión general de las opciones, consulte [integración de Office 365 con entornos local](https://support.office.com/article/263faf8d-aa21-428b-aed3-2021837a4b65.aspx).
