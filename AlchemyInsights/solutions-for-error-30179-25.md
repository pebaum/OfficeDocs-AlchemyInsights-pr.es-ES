# <a name="solutions-for-error-30179-25"></a>Soluciones para el error 30179 25

Es posible que vea un error con el mensaje "no se ha podido en secuencia de Office" (código de Error [30179 25](https://support.office.com/article/e40d3c7d-98f6-4284-94a0-882beaa44593?wt.mc_id=Alchemy_ClientDIA)) si su unidad de disco duro se está quedando sin espacio. 
  
[Liberar espacio en el disco duro](https://support.microsoft.com/help/12425/windows-10-free-up-drive-space) y, a continuación, [volver a instalar Office](https://portal.office.com/OLS/MySoftware.aspx).
